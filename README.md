# Blood Bank Management System

A web-based Blood Bank Management System designed to manage donor information, blood inventory, requests, and administrative activities efficiently. The application helps simplify the process of storing and tracking blood availability digitally.

## Project Overview

This system provides an organized platform for handling blood bank operations. It allows users to manage donor details, maintain blood stock records, and monitor requests through a centralized interface.

## Features

### Donor Management

* Add donor information
* Update donor records
* View donor details
* Delete donor information

### Blood Inventory Management

* Add available blood units
* Track blood stock
* Update blood quantity
* Monitor blood availability

### User Functions

* Search for blood groups
* Request blood availability
* View blood details

### Admin Features

* Manage donors and blood records
* Monitor requests
* Maintain system data
* Generate reports

## Installation Steps

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Create and activate a virtual environment if required.
4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Run the application:

```bash
python app.py
```

6. Open the browser and access:

```text
http://localhost:5000
```

## Project Objective

The objective of this system is to provide a simple and efficient solution for managing blood bank activities digitally while reducing manual work and improving data handling.
